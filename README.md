# bethropolis/scoop-bucket

A [Scoop](https://scoop.sh) bucket for tools by [@bethropolis](https://github.com/bethropolis).

## Usage

```bash
scoop bucket add bethropolis https://github.com/bethropolis/scoop-bucket
```

Then install any app from this bucket:

```bash
scoop install bethropolis/<app>
```

## Apps

| App | Description | Install |
|-----|-------------|---------|
| [localgo](https://github.com/bethropolis/localgo) | LocalSend v2.1 protocol — LAN file transfer CLI | `scoop install bethropolis/localgo` |

## Contributing

Manifests in this bucket are auto-generated on release via [GoReleaser](https://goreleaser.com) and other CI tools. Please open issues or pull requests on the individual app repositories rather than here.
